# High Power Rockektry
## (W.I.P)

Currently supports locating and filtering National Association of Rocketry (NAR) and Tripoli Rocketry Association (TRA) launch sites in the United States.

### Planned
- All NAR and TRA clubs
- CRA clubs
- Club pages with launch site information
- Detailed weather per launch site (cloud base, ceiling, etc.)

### Maybes
- Support for displaying launch dates
  - Webcrawler to scrape data
  - Potential user and UI system to update/add information
